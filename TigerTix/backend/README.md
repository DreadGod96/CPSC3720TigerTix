# Setting up Gemini
- Navigate to this link: https://aistudio.google.com/apikey
- Use a google account to create free api key, process should be simple
- Create .env file in backend/llm-service/.env
- Put this in the .env:
```txt
GEMINI_API_KEY=...
```
- Replace ...'s with the api key
